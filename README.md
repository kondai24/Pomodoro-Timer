# Pomodoro-Timer
ポモドーロタイマーをFlask(python)で実装しました。

【仕様】
settingで1回のポモドーロの時間と、短休憩時間と、
長休憩時間と、何サイクル行うかを決めることができる。

【デプロイしたサイトのURL】
https://pomodoro-timer-sand-six.vercel.app/
